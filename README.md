# Data Science & Big Data – Academic Projects Portfolio

## Overview

This repository contains the practical projects, statistical analyses, and machine learning pipelines developed throughout the **Data Science & Big Data** course.

The activities were designed to provide both theoretical foundations and hands-on experience with the complete data science workflow, covering data exploration, statistical inference, probability theory, predictive modeling, and machine learning optimization using Python and the scientific computing ecosystem.

Throughout the course, multiple real-world datasets were analyzed, including:

- Employee Hourly Wages
- Atlas Brasil 2014 Socioeconomic Indicators
- Advertising ROI
- Restaurant Tips
- Customer Churn Dataset

The objective was to develop analytical thinking, statistical reasoning, and machine learning skills while applying industry-standard methodologies used in modern data science projects.

---

# Topics Covered

## 1. Exploratory Data Analysis (EDA)

The first stage of the course focused on understanding data through descriptive statistics and exploratory analysis.

### Main Concepts Studied

- Measures of Central Tendency
  - Mean
  - Median
  - Mode

- Measures of Dispersion
  - Variance
  - Standard Deviation
  - Coefficient of Variation (CV)

- Quantiles and Percentiles
  - Quartiles
  - Deciles
  - Percentile Analysis

- Distribution Shape Analysis
  - Positive Skewness
  - Negative Skewness
  - Symmetric Distributions

### Practical Applications

Using workforce and socioeconomic datasets, statistical summaries were generated to identify:

- Population profiles
- Income distribution patterns
- Educational characteristics
- Workforce demographics
- Regional socioeconomic inequalities

---

## 2. Statistical Visualization

A major component of the discipline involved transforming numerical information into visual insights.

### Visualization Techniques

- Histograms
- Pie Charts
- Boxplots
- Scatterplots
- Scatterplot Matrices
- Regression Visualizations

### Objectives

Visual analysis was used to:

- Identify data distributions
- Detect outliers
- Compare groups
- Evaluate variability
- Investigate relationships between variables
- Support feature selection decisions

### Key Findings

The analyses revealed:

- Positively skewed workforce distributions
- Gender-based wage differences
- Marital status effects on earnings
- Class imbalances in categorical variables
- Regional socioeconomic disparities

---

## 3. Correlation Analysis & Predictive Relationships

The course expanded from descriptive analysis toward understanding how variables interact.

### Topics Covered

- Covariance
- Pearson Correlation Coefficient
- Linear Relationships
- Multivariate Analysis
- Regression Visualization

### Datasets Studied

#### Advertising ROI

Evaluation of advertising channels (TV, Radio, Newspaper) and their impact on sales performance.

#### Restaurant Tips

Analysis of the relationship between bill totals and tip amounts through linear regression.

#### Atlas Brasil 2014

Multivariate exploration of socioeconomic indicators using scatterplot matrices and correlation analysis.

### Skills Developed

- Relationship detection
- Predictor identification
- Multicollinearity assessment
- Interpretation of correlation coefficients
- Data-driven decision making

A dedicated module introduced probabilistic reasoning and statistical modeling.

### Discrete Probability Distributions

Applications involving:

#### Poisson Distribution

Applications involving:

- Event occurrence rates

### Continuous Probability Distributions
